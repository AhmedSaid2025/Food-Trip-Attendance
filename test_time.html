Test Time Override

لتجربة النظام مع أوقات مختلفة، أضف هذا الكود في بداية دالة getLocalTime():

// للاختبار - تغيير الوقت مؤقتاً
const testTime = "02:30"; // غير هذا الوقت للاختبار
if (testTime) {
  const [hours, minutes] = testTime.split(':').map(Number);
  const testDate = new Date();
  testDate.setHours(hours, minutes, 0, 0);
  
  // باقي الكود مع استخدام testDate بدلاً من now
  const adjustedTime = testDate;
  // ... باقي الكود
}

أمثلة للاختبار:

  • 02:30 - لاختبار الشفت الثالث
  • 07:30 - لاختبار نهاية الشفت الثالث
  • 08:30 - لاختبار بداية اليوم التشغيلي الجديد
  • 23:30 - لاختبار الشفت الثاني
